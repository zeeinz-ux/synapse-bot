Cara Kerja

Invite Bot

Klik tombol "Tambahkan ke Server" dan pilih server Discord kamu. Hanya butuh beberapa detik.

2

Login Dashboard

Login dengan akun Discord kamu. Hanya admin atau manager server yang dapat mengakses pengaturan.

3

Atur & Nikmati

Kustomisasi fitur sesuai kebutuhan server dari dashboard yang intuitif. Semua perubahan langsung aktif.

Commands Reference

Jelajahi semua perintah yang tersedia di Hidden Hamlet.

Menampilkan semua perintah
🎵

Musik

/play Putar lagu dari YouTube atau Spotify
/queue Lihat daftar antrian lagu
/skip Lewati lagu yang sedang diputar
/loop Ulangi lagu atau antrian
/shuffle Acak urutan antrian lagu
/stop Hentikan pemutaran dan keluar voice
🤖

AI Chat

/ai chat Mulai percakapan dengan AI
/ai personality Ubah kepribadian AI (friendly, formal, tsundere, wise)
/ai reset Reset konteks percakapan AI
/ai model Pilih model AI (Gemini, OpenRouter, Groq)
🛡️

Moderasi

/kick Keluarkan member dari server
/ban Ban member dari server
/mute Bisukan member sementara
/warn Berikan peringatan ke member
/clear Hapus pesar massal di channel
⚙️

Utility

/help Tampilkan daftar bantuan perintah
/stats Lihat statistik bot dan server
/welcome setup Konfigurasi welcome banner
/donation track Lacak donasi dan boost server
/autoresponder Kelola auto-reply dan trigger

About Hidden Hamlet

Kenali lebih dekat bot Discord serbaguna yang siap membantu komunitas kamu.

Hidden Hamlet

Discord Bot

🎯 Visi

Hidden Hamlet dibangun dengan visi menjadi tulang punggung komunitas Discord — menyediakan satu bot serbaguna yang menggabungkan hiburan berkualitas, kecerdasan buatan, manajemen server yang elegan, dan pengalaman pengguna yang seamless. Kami percaya setiap server Discord berhak memiliki bot yang powerful tanpa harus membayar atau mengorbankan kualitas.

👤 Developer

Dikembangkan oleh seorang enthusiast yang passionate tentang automation dan komunitas online. Dengan pengalaman dalam ekosistem Discord dan teknologi modern, Hidden Hamlet dirancang untuk scalable, reliable, dan mudah dikustomisasi oleh siapa saja — tanpa perlu jadi programmer.
